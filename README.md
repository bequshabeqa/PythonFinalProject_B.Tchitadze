# 🎬 Beka's Movies - Django Final Project

ეს არის ფილმების ბიბლიოთეკის ვებ-აპლიკაცია, რომელიც შექმნილია Django-ს გამოყენებით. პროექტი მოიცავს ფილმების დახარისხებას კატეგორიების მიხედვით, კომენტარების სისტემას, მომხმარებლის ავტორიზაციას და IMDb-დან ფილმების ინფორმაციის წამოღების ფუნქციონალს.

## 🚀 ლოკალურად გაშვების ინსტრუქცია

**1. ვირტუალური გარემოს შექმნა და ბიბლიოთეკების ინსტალაცია**
პროექტი იყენებს `uv`-ს. დააინსტალირეთ ყველა საჭირო ბიბლიოთეკა (`cinemagoer`, `Django` და ა.შ.):
`uv venv`
`uv pip install -r requirements.txt`

**2. მიგრაციების გაშვება (ბაზის შესაქმნელად)**
`python manage.py makemigrations`
`python manage.py migrate`

**3. ფილმების მონაცემების წამოღება**
ბაზის საწყისი ფილმებით შესავსებად გაუშვით:
`python manage.py fetch_movies`

**4. ლოკალური სერვერის ჩართვა**
`python manage.py runserver`
ვებ-საიტი ხელმისაწვდომი იქნება მისამართზე: http://127.0.0.1:8000/
